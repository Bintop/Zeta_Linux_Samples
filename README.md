# Zeta_Linux_Samples

#### 项目介绍
Zeta是运行于S3(ARM Cortex-A7)的产品化嵌入式Linux系统，这个项目主要同步可以在Zeta SDK上运行的应用程序示例代码，以便于开发者在S3上进行开发和定制。

#### 使用说明

1. 每个目录均代码完整的Zeta App工程，把合适的工程拷贝到Zeta SDK根目录的app里。
2. 执行zmake [dir]/zeta.in （[dir]指的是Sample工程的目录）。
3. 同步到设备上执行。
